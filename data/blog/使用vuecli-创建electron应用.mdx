# 快速开始

## 安装 vue cli

全局安装

```sh:
npm install -g @vue/cli
# 或者
yarn global add @vue/cli
```

如果升级全局的包

```sh:
npm update -g @vue/cli

# 或者
yarn global upgrade --latest @vue/cli
```

### 安装 vue-cli-plugin-electron-builder

两种安装方式

```sh:1.通过 shell 命令
vue add vue-cli-plugin-electron-builder
```

```sh:2.通过可视化界面
 vue ui
```
