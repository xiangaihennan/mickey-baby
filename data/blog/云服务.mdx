# 腾讯云

### 云开发 CloudBase

- 支持的模版较多 （框架）
- 提供用户管理看板
- 数据库 （文档形数据库） MongoDB
- 云存储 提供静态资源存储 有 cdn 加速
- 云函数 （核心功能）
- 静态网站托管
- 支持监控告警通知
- 日志查看
- 扩展能力
  ![云开发扩展能力](http://1259397000.vod2.myqcloud.com/68dd9606vodcq1259397000/8fc30d43387702292527388993/Y3JnjwQTzbEA.png)

> 使用体验：功能比较齐全，市面常用框架基本都支持；
> CloudBase CLi 还有一些不是很好用

# 轻服务 （字节）

`免费资源：可支持约 50 万次服务调用、 50 万次数据库操作和 10GB 文件存储`

### 相比腾讯云：

- `文档特别友好!!!这个很关键!!!`
- 支持创建测试环境的实例。
- 支持添加合作者 可以一起管理实例。
- 模板少的可怜

### 传统云服务 VS 轻服务

传统云服务：开发者需要自行配置计算资源，并且自行维护和管理。

轻服务：开发者可以使用轻服务提供的各项基础能力开发业务应用，开发者无需管理基础设施，即可完成业务开发。
